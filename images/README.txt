# 📸 Images Guide — Ubra Events & Boutique

Put all your image files in this `images/` folder.
The website expects these exact filenames:

─────────────────────────────────────────────────────
  FILE NAME              WHERE IT APPEARS       RECOMMENDED SIZE
─────────────────────────────────────────────────────
  logo.jpg             → Nav bar, Hero, Footer   400×400px (square)
  package-classic.jpg  → Classic Package card    800×500px (landscape)
  package-allin.jpg    → All-In Package card     800×500px (landscape)
  package-premium.jpg  → Premium Package card    800×500px (landscape)
  gallery-1.jpg        → Gallery strip photo 1   800×600px
  gallery-2.jpg        → Gallery strip photo 2   800×600px
  gallery-3.jpg        → Gallery strip photo 3   800×600px
  gallery-4.jpg        → Gallery strip photo 4   800×600px
  about.jpg            → "Why Us" section         600×800px (portrait)
─────────────────────────────────────────────────────

TIPS:
• JPG works best for photos (smaller file size).
• PNG is fine too — just rename accordingly in index.html.
• Keep file sizes under 500KB each for fast loading.
  Use https://squoosh.app to compress if needed.
• You can use different filenames — just update the
  src="images/YOUR-FILE.jpg" in index.html to match.
